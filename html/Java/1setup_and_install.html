Die Benötigten Materialien

Um mit Java starten zu können, brauchen wir zuerst eine VM (Virtual Machine) die sogenannte JRE (Java Runtime Environment), eine IDE, wir benutzen JetbrainsIntelliJ Ultmate (Hinweis: Man muss sich mit seinem Schulkonto einloggen bevor man die IDE runterlädt), jedoch kann man auch die Community Edition von IntelliJ benutzen oder man verwendet Eclipse. Zu guter letzt braucht man noch einen Compiler, das sogenannte JDK (Java Devolpment Kit).
Nun muss man nur noch den Anweisungen der Installation folgen und schon hat man alles installiert.
