Arrays

Während Variablen nicht-dimensional sind und nur einen einzigen Wert speichern können, können Arrays mehrere Werte eines Datentyps umfassen. Grundsätzlich kann ein Array ein- oder mehrdimensional sein. Zuerst befassen wir uns nur mit eindimensionalen Arrays. Wichtig ist, dass alle Elemente eines Arrays dem selbem Datentyp angehören und die Länge des Arrays konstant sein muss.

Eindimensionale Arrays

Eindimensionale Arrays können auf 2 verschiedene Arten definiert werden:

// Die Länge des Arrays wird definiert anhand der Anzahl der Werte:
Datentyp Arrayname[] = {Wert1, Wert2, Wert3, ...};

// Die Länge wird manuell definiert, die Werte müssen auch manuell initialisiert werden
Datentyp Arrayname[Länge];

Zugriff auf Arrays

Um auf die einzelnen Werte bzw. Elemente eines Arrays zuzugreifen muss zuerst der Begriff 'Index' erklärt werden: Jedes Element in einem Array hat einen sog. Index, dies ist eine Zahl die uns es erlaubt auf einen bestimmten Wert im Array zuzugreifen. Jeder Array beginnt beim Index 0, der Index wird dann für jedes Element um 1 erhöht. Das heißt, dass das erste Element des Arrays den Index 0 erhält, das zweite Element den Index 1, das dritte Element den Index 2, usw. Der Zugriff auf Arrays wird im folgenden Beispiel geschildert:

int zahlen[] = {2, 4, 8, 16, 32};
int zahl1 = zahlen[1]; // Es wird auf das 2te Element (Index 1) zugegriffen
int zahl2 = zahlen[4]; // Es wird auf das 5te Element (Index 4) zugegriffen
// 'zahl1' hat nun den Wert 4 und 'zahl2' den Wert 32
zahlen[1] = 6; // Ändert den Wert des 2ten (Index 1) Elements zu 6
int zahl1 = zahlen[1]; // 'zahl1' hat nun den Wert 6
