Arithmetische Operatoren

  • + ... Addition
  • - ... Subtraktion
  • * ... Multiplikation
  • / ... Division
  • % ... Modulo Operation

Zuweisungs Operatoren

  • = ... Zuweisung
  • += ... Additive Zuweisung
  • -= ... Subtraktive Zuweisung
  • *= ... Multiplikative Zuweisung
  • /= ... Dividierende Zuweisung
  • %= ... Modulo Zuweisung

Beispiele:

int zahl = 10;

int zahl = 0;
zahl += 5;
// entspricht: zahl = zahl + 5

int zahl = 20;
zahl -= 10;
// entspricht: zahl = zahl - 10

int zahl = 5;
zahl *= 3;
// entspricht: zahl = zahl * 3

int zahl = 100;
zahl /= 10;
// entspricht: zahl = zahl / 10

int zahl = 5;
zahl %= 3;
// entspricht: zahl = zahl % 3

Logische Operatoren

  • == ... ist gleich
  • != ... ist ungleich
  • < ... kleiner als
  • > ... größer als
  • <= ... kleiner/gleich
  • >= ... größer/gleich

Logische Verknüpfungen

  • && ... logisch UND (AND)
  • || ... logisch ODER (OR)
  • ! ... logisch NICHT (NOT)

Bitweise Verknüpfungen

  • & ... bitweise UND (AND)
  • | ... bitweise ODER (OR)
  • ~ ... bitweise NICHT (NOT)
  • ^ ... bitweise XOR
