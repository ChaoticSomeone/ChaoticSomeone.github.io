Arrays

Während Variablen nicht-dimensional sind und nur einen einzigen Wert speichern können, können Arrays mehrere Werte eines Datentyps umfassen. Grundsätzlich kann ein Array ein- oder mehrdimensional sein. Zuerst befassen wir uns nur mit eindimensionalen Arrays. Wichtig ist, dass alle Elemente eines Arrays dem selbem Datentyp angehören müssen und zusätzlich muss die Länge des Arrays konstant sein.

Eindimensionale Arrays

Eindimensionale Arrays können auf 2 verschiedene Arten definiert werden:

// Die Länge des Arrays wird definiert anhand der Anzahl der Werte:
Datentyp Arrayname[] = {Wert1, Wert2, Wert3, ...};

// Die Länge wird manuell definiert, die Werte müssen auch manuell initialisiert werden
Datentyp Arrayname[Länge];

Zugriff auf Arrays

In einem Array wird jedem Element ein sog. Index zugeordnet, dieser ermöglicht den Zugriff auf die einzelnen Elemente. Das erste Element erhält immer den Index 0 und wird für jedes Element laufend um 1 erhöht, somit hat das 2te Element den Index 1, das 3te Element den Index 2, usw:

Datentyp Arrayname[Länge];
Arrayname[Index]; // Zugriff auf den Array

Das folgende Beispiel schildert den Zugriff auf Arrays noch etwas näher:

int zahlen[] = {1, 4, 8, 16, 32};
int zahl1 = zahlen[1]; // Es wird auf das 2te Element (Index 1) zugegriffen
int zahl2 = zahlen[4]; // Es wird auf das 5te Element (Index 4) zugegriffen
// 'zahl1' hat nun den Wert 4 und 'zahl2' den Wert 32
zahlen[1] = 6; // Ändert den Wert des 2ten (Index 1) Elements zu 6
zahl1 = zahlen[1]; // 'zahl1' hat nun den Wert 6
